# Catalyst Marketing Website

Static front-end build for Catalyst Marketing LLC.

## Local preview

Run the included static server:

```bash
node serve-local.mjs
```

Then open `http://127.0.0.1:4174/`.

## Pages

- `/`
- `/services/`
- `/process/`
- `/about/`
- `/contact/`
- `/thank-you/`

The Thank You page is intentionally hidden from public navigation and marked `noindex`.
